# How to use the project

Requisitos:
-Docker
-NodeJs 14.20.0

Correr con Docker:

- Clonar repositorio en url:
- docker build . -t dockerized-react
- chequear que la image fue creada "docker images"
- correr la image de docker "docker run -p 3000:3000 -d dockerized-react"

Correr local:

- Clonar repositorio en url:
- Correr "npm install"
- Correr "npm start"
